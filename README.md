#  Water Smarter

*A simple, educational, and interactive website promoting water conservation through facts, rules, and a smart watering calendar.*

---

## Overview

**Water Smarter** is a student-friendly, browser-based project designed to help people understand:

* Why water conservation matters
* Seasonal watering rules and restrictions
* Daily plant watering reminders through an interactive calendar

The project contains **three main pages**, each focusing on a different aspect of water awareness and responsible usage.

---

## Project Pages

### 1️ **Home Page** (`index.html`)

 *Source:* 

This page introduces the urgency of water conservation with real examples:

* Lake Mead’s low water levels
* Lake Tahoe’s limited capacity
* Nevada’s delicate ecosystems

---

### 2️ **Water Laws & Rules** (`index copy.html`)

 *Source:* 

This page explains seasonal watering guidance:

*  **Spring** → 2 days/week
*  **Summer** → 4 days/week
*  **Fall** → 3 days/week
*  **Winter** → 2 days/week

And the most important rule:

>  **NEVER WATER ON SUNDAYS!**

This page visually displays seasons using images, making it easy to remember the rules.

---

### 3 **Interactive Watering Calendar** (`interactive.html`)

 *Source:* 

This is the most advanced part of the project.
It includes:

* Interactive monthly calendar
* Alarm scheduling for different plant types
* Customizable date & time input
* Optional recurring days
* Prevents watering alarms on Sundays
* Plays an alarm sound + pop-up reminder

Users can select a plant (Rose, Cactus, Orchid, Tulip), pick a date & time, and receive a reminder such as:

> “Time to water the cactus!”

## Features

### Educational

Teaches the importance of water conservation using real statistics and ecosystem examples.

### Navigation

Every page includes a shared top navigation for easy switching.

### Interactive Tool

Custom calendar with alarms to help users maintain a good watering routine.

### Visual Learning

Picture-based storytelling to engage younger audiences.

## How to Run

Just open any of the HTML files in your browser! No server required.

1. Download the project
2. Double-click `index.html`
3. Explore the site
4. Visit **Water Calendar** to try interactive features
